A Simple GUI Encrypted Notes Taking App,
It's a very simple application which doesn't seems like a note taking application at all but it does in disguise.

**How To Use**
In the Use Me! text box, just save your notes in this format <password>, < your secret notes > for example if i want to save all of my notes with the secret 
passcode : 4029, then i will enter this in the input field : 4029, Helllo where Helllo is my "secret note".

Keep adding as much notes as you like but there is a condition that "You can't change your password from the one you set while writing your first note
and you have to keep saving all of your remaining notes with the same password if you wish to use a new password then just remove the 'enc.txt' file
from the project directory and all of your past saved notes will be gone allowing you to use a new password.


If you wish to access you notes,
Click the checkbox and From the drop down menu at the top, Click "Access Notes", and enter your password.


Remember :
No one else can access your notes without the password and this password is saved nowhere else !
